URL Exclusions

How URL exclusions work:
  • Enter one URL per line
  • Use wildcards (*) for pattern matching (e.g., *.example.com)
  • Add comments using # at the beginning of the line
  • Both full URLs and domain names are supported
  • WebRTC blocking will be disabled on matching URLs
